# AI Trainer — Автоматизация спортивного планирования с использованием ИИ

Адаптивная веб-платформа, использующая искусственный интеллект для персонализированного планирования тренировок, питания и отслеживания прогресса спортсменов.

> Исследовательская работа по направлению "Искусственный Интеллект"

## Описание проекта

AI Trainer решает проблему универсальных тренировочных программ, которые не учитывают индивидуальные параметры спортсмена. Платформа генерирует персонализированные планы тренировок и питания на основе физиологических данных, медицинских ограничений, целей и доступного оборудования пользователя.

### Ключевые возможности

- **ИИ-генерация тренировок** — персонализированные программы с учётом уровня подготовки, целей, оборудования и медицинских ограничений
- **Чат с ИИ-тренером** — контекстно-зависимый диалог с учётом профиля пользователя (стриминг ответов)
- **Аналитика и прогресс** — графики динамики веса, объёма тренировок, история занятий
- **Календарь тренировок** — распределение занятий по дням с отслеживанием выполнения
- **Библиотека упражнений** — 90+ упражнений с описаниями, фильтрацией по мышцам/оборудованию

## Быстрый старт

### Требования

- Python 3.12+
- Node.js 20+
- PostgreSQL 16+
- Redis 7+

### Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Настройка окружения
cp ../.env.example .env
# Заполнить .env (DATABASE_URL, OPENAI_API_KEY, SECRET_KEY)

# Миграции
alembic upgrade head

# Заполнение БД упражнениями и продуктами
python -m app.scripts.seed_db

# Запуск
uvicorn app.main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Фронтенд будет доступен на `http://localhost:3000`, API — на `http://localhost:8000`.

### Docker

```bash
docker compose up -d
```

## API

Документация API доступна после запуска бэкенда:
- Swagger UI: `http://localhost:8000/docs`
- ReDoc: `http://localhost:8000/redoc`

### Основные эндпоинты

| Метод | Путь | Описание |
|---|---|---|
| `POST` | `/api/v1/auth/register` | Регистрация |
| `POST` | `/api/v1/auth/login` | Авторизация |
| `GET/PUT` | `/api/v1/profiles/me` | Профиль пользователя |
| `POST` | `/api/v1/workouts/generate` | Генерация плана тренировок (ИИ) |
| `POST` | `/api/v1/workouts/schedule/start` | Начало тренировки |
| `POST` | `/api/v1/workouts/schedule/{id}/sets` | Логирование подхода |
| `POST` | `/api/v1/chat/conversations` | Создание диалога с ИИ |
| `GET` | `/api/v1/analytics/dashboard` | Сводная аналитика |
| `GET` | `/api/v1/exercises/` | Библиотека упражнений |
